# Actiance Hack

POC with our partner Actiance

## Getting Started

```
g clone https://github.com/User1m/actiance-hack.get
```

### Prerequisites

Visual Studio or Visual Studio for mac

### Installing

Open the solution:

```
App.sln
```

Run project in Visual Studio

## Running the tests

```
Build App.Tests
```

```
Run App.Tests
```
